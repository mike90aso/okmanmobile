# Port St. Lucie DOT Physicals

Static website for DOT physicals in Port St. Lucie, FL.

## Deploy to Netlify

1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → Add new site → Import from Git
3. Connect GitHub and select this repo
4. Netlify will auto-detect the static site (publish directory: `.`)
5. Deploy — future pushes to `main` will auto-deploy
